# ToDo App

タスクの追加・削除・完了チェックができる、1ファイル完結のシンプルなToDoリストアプリです。HTML / CSS / JavaScript をすべて `index.html` にまとめています。

## 使い方

1. 入力欄にやることを入力し、「追加」ボタン（または Enter キー）でリストに追加します。
2. タスク左側のチェックボックスをオンにすると、完了として取り消し線が入ります。
3. 各タスクの「削除」ボタンで、そのタスクだけを消せます。

ローカルで使う場合は `index.html` をブラウザで開くだけで動きます。
